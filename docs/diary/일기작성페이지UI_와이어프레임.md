# WhisperMind 일기 작성 페이지 와이어프레임

## 1. 메인 흐름 개요

WhisperMind 앱의 일기 작성 기능은 사용자가 자신의 감정과 생각을 쉽고 편리하게 기록할 수 있도록 설계되었습니다. 아래는 주요 사용자 흐름을 와이어프레임으로 시각화한 것입니다.

## 9. 사용자 경험 핵심 포인트

1. **단계적 접근**
   - 일기 작성의 주요 단계(감정 선택 → 제목 → 본문)를 순차적으로 안내
   - 각 단계가 명확하게 구분되어 사용자가 자연스럽게 진행

2. **감정 중심 디자인**
   - 감정 선택이 가장 상단에 배치되어 감정 표현의 중요성 강조
   - 감정에 따라 미묘하게 변화하는 UI 테마 색상으로 감정적 몰입감 향상

3. **직관적인 도구 배치**
   - 가장 자주 사용하는 기능(미디어 추가, 태그)을 언제든 접근 가능하도록 배치
   - 모바일에서는 하단에, 태블릿/데스크톱에서는 좌측 패널에 배치

4. **AI 지원 기능**
   - 글쓰기를 돕는 AI 기능을 필요할 때만 호출하여 자연스러운 작성 흐름 방해 최소화
   - 작성 중인 내용에 맥락을 이해한 제안으로 더 풍부한 표현 지원

5. **맥락 보존 자동 저장**
   - 주기적 자동 저장으로 작성 내용 손실 방지
   - 저장 상태를 항상 표시하여 사용자에게 안정감 제공

이 와이어프레임은 사용자가 감정을 중심으로 자신의 생각과 경험을 편리하게 기록할 수 있도록 최적화되었습니다. 직관적인 인터페이스와 단계적 접근법을 통해 일기 작성의 진입 장벽을 낮추면서도, 다양한 미디어와 AI 지원 등 풍부한 표현 도구를 제공합니다.
